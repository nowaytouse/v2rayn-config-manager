# 更新日志

## [2.0.0] - 2024

### 🎉 重大更新

#### 新增功能
- ✨ **GUI 模式**: 全新的图形用户界面，基于 Fyne 框架
  - 可视化订阅管理
  - 实时日志显示
  - 配置界面化编辑
  - 一键更新操作
  
- 🖥️ **多平台支持**: 完整支持 Windows、Linux、macOS
  - 自动检测系统平台
  - 平台特定的路径处理
  - 适配不同系统的权限机制
  
- 📦 **模块化架构**: 重构代码结构
  - 业务逻辑分离到 `pkg/core`
  - GUI 逻辑独立到 `pkg/gui`
  - 更好的代码组织和维护性
  
- 🔧 **双模式运行**: 支持 GUI 和 CLI 两种运行模式
  - `-gui` 参数启动图形界面
  - 默认命令行模式保持向后兼容
  - 灵活适应不同使用场景

#### 改进
- 🎯 **更好的错误处理**: 统一的错误处理和日志记录
- 📝 **完善的文档**: 新增多个文档
  - README.md - 完整的项目说明
  - QUICK_START.md - 快速开始指南
  - ARCHITECTURE.md - 架构说明文档
  - CHANGELOG.md - 本文档
  
- 🚀 **构建系统**: 多种构建方式
  - `build.sh` - Linux/macOS 构建脚本
  - `build.bat` - Windows 构建脚本
  - `Makefile` - Make 构建支持
  
- 🔒 **配置管理**: 改进的配置文件处理
  - 自动生成默认配置
  - 配置验证
  - 保存/加载功能

#### 技术栈
- Go 1.21+
- Fyne v2.4.5 (GUI 框架)
- 纯 Go 标准库（核心功能）
