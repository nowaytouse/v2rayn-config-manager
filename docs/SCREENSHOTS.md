## 命令行模式

### 帮助信息

```bash
$ ./singbox-manager -help
Sing-box Manager - 配置和核心自动更新工具

用法:
  singbox-manager [选项]

选项:
  -gui               启动图形界面模式
  -config <path>     指定配置文件路径 (默认: config.json)
  -once              仅执行一次更新任务后退出（命令行模式）
  -version           显示版本信息
  -help              显示此帮助信息

示例:
  singbox-manager -gui                    # 启动 GUI 模式
  singbox-manager                          # 命令行模式，按配置定时更新
  singbox-manager -once                    # 仅执行一次更新
  singbox-manager -config custom.json      # 使用自定义配置文件

配置文件示例请参考 config.json
```

## 特性演示

### 多订阅管理
- 支持同时管理多个订阅源
- 每个订阅独立配置名称、URL 和保存路径
- 可视化的订阅列表

### 实时日志
- 所有操作都有详细日志记录
- 时间戳精确到秒
- 支持清空日志功能

### 一键更新
- 订阅更新
- 核心更新  
- 全部更新（组合操作）

### 自动定时更新
- 可配置更新间隔（小时）
- GUI 模式下可以启动/停止自动更新
- CLI 模式自动按配置运行

## 平台特定界面

### Windows
- 原生 Windows 风格
- 支持系统主题
- 任务栏图标

### macOS
- 原生 macOS 外观
- 支持深色模式
- Dock 图标

### Linux
- GTK3 主题支持
- 适配各种桌面环境（GNOME、KDE、XFCE 等）
- 系统托盘图标（如果桌面环境支持）
